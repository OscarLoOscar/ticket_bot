🛠️ 常見問題 FAQ
Q: 深色模式下 UI 空白？
A: macOS 的原生 ttk 深色支援不完整，已改為強制黑底白字。如仍無法顯示，請切換為「淺色」再測試。

Q: 為何登入沒反應？
A: 確認網頁欄位的 name="username" 與 name="password" 是否正確，如不符請手動登入。

Q: 驗證碼無法自動處理？
A: 目前僅提示使用者手動輸入，可加裝 OCR 模組進一步辨識（選配）。
